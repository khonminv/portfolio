6) jQuery 대상


6-1) CSS 기법

	background-color: #f90;
}


6-2) jQuery 기법

$(".html").css({
	"background-color": "#f90"
});


$ : jQuery 시그널

입력 데이터 형식 : 문자, 숫자, 블린 유형

문자 형식은 큰 따옴표나 작은따옴표로 입력해야 합니다.

$를 사용하면 jQuery 객체가 만들어집니다.

객체는 속성과 명령어가 있는 주체적인 대상을 뜻합니다.
