1) 오늘 배운 JavaScript


1-1) console.log()

인자 내용을 출력합니다. console 탭을 통해 볼 수 있습니다.



2) 오늘 배운 jQuery


2-1) $()

인자 내용은 문자 형식으로 'CSS 기법'을 작성합니다.


2-2) css()

인자 내용은 {속성: 값, 속성: 값} 방식으로 작성됩니다.


2-3) addClass(), removeClass()

인자 내용은 클래스 이름 문자입니다.


2-4) click(function(){})

인자 내용은 명령 내용입니다.



3) 연습 예제


$("#tutorial li").click(function(){
	// console.log("click event");

	$("#tutorial li").removeClass("active");
	$(this).addClass("active"); // $(this) 클릭된 이벤트 대상입니다.
});
