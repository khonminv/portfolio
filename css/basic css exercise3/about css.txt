height: 100vh;

윈도우의 전체 세로 크기입니다.


min-height: 600px;

minimum height : 최소 크기

최소 크기로 600px 세로를 가집니다.

600px 이상입니다.


max-width: 720px;

maximum width : 최대 크기

최대 크기로 720px 가로를 가집니다.

720px 이하입니다.


object-fit: cover;

object-fit 컨테이너에 들어있는 이미지 스타일입니다.

이미지를 컨테이너에 빈 영역없이 꽉 채웁니다.


#header {
	/* display: block; */
	position: absolute;
}

컨텐츠 크기로 가로 크기가 지정됩니다.

absolute로 스타일하면 display와 상관없이 컨텐츠 크기로 만들어지는 block으로 생각하시면 됩니다.


transform CSS

변형을 관리하는 스타일입니다.

현재의 스타일을 기본 정보로 가지고 있습니다.

transform: rotate(90deg);

transform: scale(2);

transform: translateX(-10px);

transform: translateX(-50%);

transform: translate(-50%,-50%); /* x, y */
