1) display


1-1) block

Box Model 모두 지원되는 스타일입니다.

div, h1 ~ h6, p, ul, li, ...


Box Model은 양감을 표현하는 스타일 방법입니다.

width, height, margin, padding, border


1-2) inline

span, a, ...

기본 크기는 텍스트 크기이고, 크기를 지정할 수 없습니다.


1-3) inline-block

img, input, textarea, ...

기본 크기는 텍스트 크기이고, 크기를 지정할 수 있습니다.


1-4) flex


1-5) position

포토샵의 레이어처럼 배치하는 원리입니다.

UI 따로 학습할 필요가 있습니다.

모든 요소는 static으로 정의됩니다.


1-5-1) position: static;

left right top bottom z-index 속성 스타일이 불가능합니다.


1-5-2) position: relative;

absolute의 컨테이너 역할을 할 수 있습니다.

static 포지션처럼 스타일을 할 수 있습니다.

.box {
	position: relative;
	margin-top: 35px;
}

absolute 포지션처럼 스타일을 할 수 있지만, 부피를 가지고 있습니다.

.box {
	position: relative;
	top: 35px;
}


left right top bottom z-index 속성 스타일이 가능합니다.


1-5-3) position: absolute;

절대위치로 배치됩니다.

left right top bottom z-index 속성 스타일이 가능합니다.


1-5-4) position: fixed;

left right top bottom z-index 속성 스타일이 가능합니다.

absolute와 유사하지만, 윈도우 위치 기준으로 정렬됩니다.


1-6) 가상요소

:hover	

마우스 올린 가상요소입니다.


:focus

탭한 가상요소입니다.


:first-child

첫 번째 리스트입니다.


:last-child

마지막 리스트입니다.


:nth-child()

순번 리스트입니다.


::before

이전 디자인 가상요소입니다.


::after

이후 디자인 가상요소입니다.


1-7) opacity와 display

opacity: 0;

불투명도 0%, 자취는 남아 있습니다.


display: none;

안보이는 스타일입니다. HTML은 남아 있습니다.
